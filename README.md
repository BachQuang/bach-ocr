To use train.py:
  - Create data folder 
  - Put vocab.json to ./data
  - Put your data to ./data
  - Data folder should have:
    - train.json = {'path_to_text_line_image_1': label1,
                     'path_to_text_line_image_2': label2.....}
    - images folder: contain textline images
 
 Test on SROIE dataset: 
  - acc_by_char ~ 98%
  - acc_by_word ~95%
